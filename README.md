# Akayo
Arknights 启动，作战，收菜，搓玉一脚本解决，适用于没有时间的学生~~放弃游戏~~

## 功能
* [X] 启动游戏
* [X] 清空理智
* [X] 提交任务
* [X] 便捷的拓展 API
* [X] 多分辨率适配(依赖OCR 与OpenCV识别，理论上分辨率越大越好)
* [X] 自动剿灭
* [ ] 自动检测是否停止剿灭
* [X] 自动搓玉
* [ ] 基建轮班
* [ ] 自动招募(等待上游Auto.js修复bug)
* [X] 适配官服
* [X] 收菜
